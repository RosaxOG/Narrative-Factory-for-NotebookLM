# Narrative Factory for NotebookLM

## Overview
Narrative Factory for NotebookLM is a project designed to facilitate the creation and management of narratives in a structured, efficient manner, specifically tailored for NotebookLM.

## Features
- **Template System**: Easily create narratives from predefined templates.
- **Collaboration Tools**: Supports multi-user editing and feedback.
- **Customizable Structure**: Adjust the narrative flow according to project needs.
- **Export Options**: Export narratives in various formats including Markdown, HTML, and PDF.

## Installation Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/RosaxOG/Narrative-Factory-for-NotebookLM.git
   cd Narrative-Factory-for-NotebookLM
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the application**:
   ```bash
   npm start
   ```
